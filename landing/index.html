Everything your AI tools built — in one report

whatidid is read-only, local-first, and credential-safe. It never touches your tokens, keys, or passwords.

🔍

Cross-agent coverage

Reads Hermes, OpenAI Codex CLI, Claude Code, GitHub Copilot CLI, and Continue — all in one pass.

💰

Value estimation

Converts session data into estimated human-equivalent hours and dollar value. Show your team what AI is actually worth.

📊

HTML + Markdown + JSON

Self-contained HTML report with a Linear-inspired design. Also outputs structured JSON for dashboards and CI pipelines.

🔒

Credential-safe

Hardcoded redaction of API keys, tokens, passwords, and private key patterns before any text hits the report.

🗂️

Project grouping

Sessions are normalized to your actual repos and projects. Work is grouped into human-readable workstreams.

⚡

GitHub Action ready

Drop the included action into any repo for automatic weekly digests uploaded as build artifacts.

Get started in 30 seconds

Install from PyPI, run against your last 7 days, open the report.

$ pip install whatidid
$ whatidid --days 7

Or use the GitHub Action

# .github/workflows/whatidid.yml
uses: th0mps0nty/whatidid/action@v1
with:
  days: '7'
  hourly-rate: '125'
